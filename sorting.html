Sorting Algorithm Visualizer

React-based visualizer of 6 of the most popular sorting algorithms
that allows to compare 2 or 4 at the same time.

Project Description

After deciding to learn React due to its growing popularity and powerful features, I decided to create this project to solidify my knowledge and skills with this library. I've always found this project particularly interesting and worthy of being implemented because it forces you to understand sorting algorithms on a deeper level, which are very common in computer science. For some reason, I always thought of this web application as something really hard to implement. Before I started the implementation, I had no idea how I would do it and it seemed like a fun challenge. So, I decided to give it a shot.

I chose 6 algorithms that I thought were the most popular ones.

  • Bubble Sort
  • Selection Sort
  • Insertion Sort
  • Merge Sort
  • Heap Sort
  • Quick Sort
Upon loading, the page shows a randomly generated array. The user has the option of visualizing a single algorithm, comparing 2 algorithms, or comparing 4 algorithms. The user can also generate another random array at any point in time. There's a drop-down element where users select the algorithm below each copy of the array. All elements are created with React. The UI is pretty straightforward and it allows users to focus on the algorithm visualization. The 2 elements being compared at any point in time are shown in red.
