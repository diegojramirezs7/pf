to design and implementation, and finally deployment and maintenance.
The following projects allowed me to develop a solid
understanding of different technology stacks by creating clean
and innovative solutions that solve challenging problems.

Voice Recognition and
Authentication System

Hands-free voice authentication system built for a local company that wanted to transition from username and password system to voice only identification and verification of employees and guests.

Sorting Algorithm
Visualizer

React-based sorting algorithm visualizer. This web application allows to visualize 1, 2 or 4 algorithms at a time for comparison. It includes 6 of the most popular sorting algorithms.

Twitter Bot

Python script that tweets/retweets content automatically every x hours. The content is obtained from either scraping a set of websites or from a preselected list of REST APIs. The program is containerized with Docker and runs on an AWS instance.

IoT System for Rodent
Population Analysis

This system seeks to improve the current method researchers use to study rodent populations by automating the data gathering process. It consists of a variety of sensors and a HD camera that collect the appropriate information when the rodent is detected.

This project consists of a food ordering and delivery system where users can track the status of the order and see the live reduced location of the delivery driver through their browser. A key focus of this project was to protect the employee's privacy from the company.
