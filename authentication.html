Voice Recognition and Authentication System

User authentication system implemented with voice identification and
verification techniques to replace the traditional username/password process.

Project Description

As part of a final project, we met with a client who wanted to have a completely hands-free authentication process for his company. So, the goal of this project was to build a complete system that would allow employees or guests to be authenticated by only using their voice. The main features to be implemented were:

  • Speech recognition (speech-to-text conversion)
  • Allow the user to say different commands (Register, Verify Me, etc)
  • Authenticating users by their voice
  • Providing a confidence rating for the authentication process
  • Providing audio feedback of results (text-to-speech conversion).

After doing extensive research, we found 3 APIs that could be perfectly integrated into our system to meet all the former requirements. For text-to-speech and speech-to-text conversion we used Google's Voice API, and for the authentication process we combined 2 APIs for extra security (Azure Cognitive Services and VoiceIt). We implemented a server that integrates all the APIs, handles requests from the client on the Raspberry Pi and the web UI, and connects to the MySQL database to retrieve and save user data. Additionally, a web interface was included for easier user management (creating, retrieving, updating, deleting users). Both the script on the RPi and the server were written in Python. Python was chosen due to the extensive set of libraries that allowed us to implement the features more easily.

Upon integration of all the components, we successfully implemented a system that satisfied all of the mentioned requirements, and some other implicit ones. After a lot of testing, we managed to create a smooth user experience in which users can register, get authenticated and give voice commands to the system while the system talks back to the through a speaker. User authentications consistently returned a confidence rating above 85%. Both our professor and our client were extremely satisfied and we got an "A as a grade".
